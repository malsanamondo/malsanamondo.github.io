Raspberry Pi: montar un servidor web en casa (3)

25-04-2014

En la primera y segunda parte de este manual hemos visto qué es la Raspberry Pi y cómo instalar Raspbian en la tarjeta SD. En esta tercera parte configuraremos el sistema operativo para que funcione como servidor web y para que podamos acceder a él desde cualquier parte.

Después de copiar Raspbian en la tarjeta SD podemos comenzar la configuración. En este punto necesitamos además de la Raspberry Pi y de la SD, el alimentador micro-usb (sirve cualquier cargador micro-usb de móvil de al menos 700 mA), un teclado usb, un monitor y un cable RJ-45 (ethernet) para conectarla al router.

Introducimos en la Raspberry Pi la tarjeta SD, la conectamos al monitor (con cable HDMI o RCA), la conectamos al router, le conectamos un teclado usb, y por último le conectamos el alimentador. La Raspberry Pi no dispone de ningún botón de encendido, basta conectarla a la corriente y automáticamente se encenderá (más adelante se explica cómo apagarla correctamente).

En el primer encendido se nos mostrará la herramienta de configuración, una ventana con algunas opciones principales. Posteriormente podemos volver a abrirla con el comando:

$ raspi-config

Para moverse por esta ventana utilizaremos las teclas de dirección y la tecla TAB para saltar de campos, para aceptar la tecla INTRO, para cancelar la tecla ESC y para seleccionar o desmarcar la barra de espacio. De las opciones que aparecen las que nos interesan ahora son:

  • Expandir el sistema: para que Raspbian ocupe todo el espacio de la tarjeta.

  • Activar inicio en modo escritorio: se puede elegir que arranque en modo gráfico o en modo consola, por defecto arranca en modo consola y es la opción con la cual trabajaremos. Una vez arrancado el sistema si queremos iniciar el modo gráfico basta con escribir:

    $ startx

  • Opciones de internacionalización: configurar idioma, localización, teclado, etc. En idioma desmarcamos (con la barra de espacio) en_GB y seleccionamos es_ES UTF8 (si lo quieres en castellano), elegimos distribución del teclado (normalmente PC genérico 105 teclas) e idioma de teclado. En las siguientes opciones no hace falta cambiar nada.

  • Opciones avanzadas: cambiamos el nombre de la máquina por uno fácilmente reconocible (nos servirá en los siguientes pasos), y activamos el acceso SSH para acceder a la Raspberry Pi de forma remota desde otros dispositivos.

Una vez realizados los cambios reiniciamos para no tener problemas con el teclado, escribimos:

$ sudo reboot

Después de reiniciarse nos pide la contraseña (que por defecto es raspberry), y procedemos a cambiarla para aumentar la seguridad de nuestro dispositivo (el cambio de contraseña se puede realizar junto a la primera configuración, pero es posible que el sistema aún no reconozca el idioma que hayas configurado, por eso, para asegurarnos lo hacemos en un segundo momento después de reiniciar). Para hacerlo escribimos:

$ raspi-config

y vamos a:

  • Cambio de contraseña: por defecto el usuario es pi y la contraseña es raspberry. Por seguridad cambiamos la contraseña.

Una vez realizado el cambio escribimos uno a uno estos comandos:

$ sudo apt-get update

$ sudo apt-get upgrade

y esperamos el tiempo necesario para que el sistema se actualice. Es conveniente repetir estos comandos cada cierto tiempo para asegurar el óptimo funcionamiento del sistema operativo.

Raspberry Pi: montar un servidor web en casa (2)

17-04-2014

Antes de continuar con este manual recomiendo la lectura de la primera parte. A continuación instalaremos el sistema operativo de la Raspberry Pi.

La Raspberry Pi no tiene disco duro ni SSD, en su lugar utiliza una tarjeta SD para almacenar los datos y es allí donde debemos instalar el sistema operativo.

Existen varios sistemas operativos para la Raspberry Pi, muchos de ellos distribuciones GNU/Linux. Aquí muestran algunos de ellos. Nuestro servidor lo montaremos con Raspbian (una versión de la distribución Debian Wheezy del sistema operativo GNU/Linux).

Raspberry Pi: montar un servidor web en casa (1)

16-04-2014

Raspberry Pi es un ordenador de placa reducida o placa única (SBC) de bajo costo, desarrollado en Reino Unido por la Fundación Raspberry Pi, con el objetivo de estimular la enseñanza de ciencias de la computación en las escuelas.

Las especificaciones las puedes consultar aquí, y en Internet puedes encontrar fácilmente información de precios (la versión B de 512 MB cuesta unos 36 €), accesorios (cajas, tarjetas SD/SDHC, cables, alimentadores, adaptadores wifi, disipadores, cámaras, etc.), sistemas operativos, usos, proyectos, etc.
